Detailed Description

Single scalar value of specified Type. Can be used in matrix and vector expressions as additional initial value or param.

Note
Can be empty, i.e. contain no value.
Can be updated from the host using ScalarDataWrite expression node.
Scalar content can be accessed from host using ScalarDataRead expression node.
