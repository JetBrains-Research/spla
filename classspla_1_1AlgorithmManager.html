Detailed Description

Manages particular algorithms implementations.

Allows to register a number of different algorithms for a particular processing task. Provides functionality to dispatch algorithm in the same thread or to create askfloat task with selected algorithm.

For example, for MxM operation of matrix blocks can store several algorithms, such as MxM_csr, MxM_coo and etc. and choose suitable for provided params.
