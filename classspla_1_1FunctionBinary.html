Detailed Description

Binary Typed element function f: a x b -> c.

Binary function accepts two values of typed input objects, with types A and B and return new value for typed object with type C. Can be used as binary op in element wise operations and as accum for matrices, vectors and scalars.

Source code for the function must be provided in the string. Function signature is following (const void* vp_a, const void* vp_b, void* vp_c), where a, b and c pointers to values, c must be written by the function. It is up to user to cast this pointers to appropriate types and check values sizes.
