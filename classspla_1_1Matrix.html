Detailed Description

Matrix object to represent a mathematical dim MxN matrix with values of specified Type. Uses blocked storage schema internally.

Note
Can be used as mask (only indices without values) if Type has zero byteSize.
Can be updated from the host using MatrixDataWrite expression node.
Matrix content can be accessed from host using MatrixDataRead expression node.

Uses sparse values storage schema, so actual values of the matrix has mathematical type Maybe Type, where non-zero values stored as is (Just Value), and null values are not stored (Nothing). In expressions actual operations are applied only to values Just Value. If provided binary function, it is applied only if both of arguments are Just Arg1 and Just Arg2.
