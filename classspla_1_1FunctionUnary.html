Detailed Description

Unary Typed element function f: a -> b.

Unary function accepts value of typed input object, with type A and returns new value for typed object with type B. Can be used as unary op in transform operations for matrices, vectors and scalars.

Source code for the function must be provided in the string. Function signature is following void(_ACCESS_A const void* vp_a, _ACCESS_B void* vp_b), where a, and b pointers to values, b must be written by the function. It is up to user to cast these pointers to appropriate types and check values sizes.
